# 🏥 Curelex Healthcare Backend System

## Project Description
Curelex Healthcare Backend System is a Spring Boot based backend application used to manage patients, doctors, appointments, and medical records securely.

---

## Tech Stack
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- File Upload
- Email Notification
- Role Based Access

---

## Modules
✔ Patient Authentication  
✔ Doctor Approval System  
✔ Appointment Booking  
✔ Medical Records  
✔ Dashboard APIs

---

## Setup Instructions

1. Clone Repository
   git clone https://github.com/Davi-Bansal/healthcare-Backend.git

2. Copy application-example.properties  
   Rename it to application.properties

3. Add database and email credentials

4. Run project
   mvn spring-boot:run
